Guida markdown

Quando si inserisce un articolo è possibile utilizzare il markdown per dare struttura al testo.

Puoi scaricare questa guida.

Header

Per inserire degli header è necessario inserire un numero di cancelletti pari al livello di header che si vuole inserire (es. 2 asterischi per h2) seguiti da uno spazio.

# This is an <h1> tag
## This is an <h2> tag
###### This is an <h6> tag

Link

Per inserire un link è necessario inserire prima il testo che si vuole visualizzare contenuto in parentesi quadre, successivamente il link effettivo contenuto tra parentesi tonde.

[testo del link visibile](https://www.google.com)

Grassetto

Per inserire testo in grassetto bisogna apporre due asterischi o due underscore all'inizio e alla fine del testo da modificare.

**This text will be bold**
__This will also be bold__

Enfasi

Per dare enfasi ad una porzione di testo inserire un carattere asterisco o un carattere underscore all'inizio e alla fine.

*This text will have emphasis*
_This text will have emphasis_

Liste

Per inserire una lista non ordinata inserire gli elementi uno per ogni riga, inserendo un asterisco all'inizio di ognuna seguito da uno spazio. Per inserire una lista ordinata inserire invece il numero seguito da un punto e uno spazio all'inizio di ogni riga.

Lista non ordinata:
* elemento
* elemento
Lista ordinata:
1. elemento 1
2. elemento 2

Citazioni

Per inserire una citazione è necessario scriverla su una riga senza altro contenuto, usare il carattere di maggiore seguito di uno spazio ed inserire la citazione.

> questa è una citazione

Lingua inglese

Per inserire una o più parole in lingua inglese aprire la parentesi tonda seguita da un punto interrogativo, inserire le parole inglesi, chiudere la parentesi.
